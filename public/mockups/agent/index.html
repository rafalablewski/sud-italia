Agent Commerce — the chosen direction
one capability layer, two channels: Concierge

Concierge is the agent-commerce direction. It reuses the POS + KDS + CRM visual language — Inter + JetBrains Mono, the Atlas dark canvas, the header/footer chrome — and leans on the system's violet "AI" accent. The thesis: as people increasingly discover and order through AI assistants (ChatGPT, Claude, Perplexity) and WhatsApp, the restaurants that expose structured, real-time data — menu, availability, allergens, ordering, payments — win the order. So Ottaviano exposes one capability layer: as MCP tools/resources any AI agent can call, and as the WhatsApp pre-order agent a human chats with. Same get_menu, get_allergens, place_order, create_payment behind both. Open the live mockup below; it's interactive end to end.
