POS — the chosen direction
selected from ten: Tabs

Tabs is the POS direction we're taking forward. It reuses the Kitchen Display System visual language — the same Inter + JetBrains Mono type, the Atlas dark canvas, the tone palette and the header/footer chrome — so POS and KDS read as one system. The thesis: a busy truck window juggles several orders at once, so the terminal puts a rail of open checks front and centre — each its own running total and status — that you switch between, park, and send to the KDS independently.
