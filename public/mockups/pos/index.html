Sud Italia · Point of Sale

POS — ten design directions
cashier order-entry, in the KDS visual language

Ten ways to build the counter-side terminal that takes the order before it hits the kitchen. Every direction reuses the Kitchen Display System design language — the same Inter + JetBrains Mono type, the same dark/light canvases, tone palette and chrome — so POS and KDS read as one system. Each tile opens a live, interactive mockup: tap a product to ring it up, quantities step, combo discounts subtract from the total, and the charge flow runs to a paid confirmation. Every one has a fullscreen toggle, a live clock, and the same eight-table Kraków menu with real prices.
