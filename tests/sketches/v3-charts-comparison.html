Why A: reads a trend in one glance (the thing tables can't show), keeps the dense Bloomberg/Linear feel, and reuses the exact inline-SVG approach already in Sparkline.tsx — so it stays inside the v2-deletion isolation contract (no Recharts, no v2/charts import). Cost: ~1 small primitive file (Chart.tsx: area/bar/donut) to build + document once, then every surface (Reports, Feedback, Multi-location, Dashboard) reuses it.
