Calendar redesigns

Ten takes on the day picker that sits in the Book toolbar — each fully interactive with previous day / next day navigation, built on the live Core theme tokens. Click through them; the readout under each shows the selected ISO date.

CURRENT  ·  a native <input type="date"> behind a static pill — no day-step arrows, OS-native popup, breaks the warm Core look. The concepts below fix all three.
01Pill + arrows

The current pill, kept — but flanked by ‹ › day-steppers and rounded to match Core’s brand buttons. Smallest change, biggest win: the arrows people were missing.

02Week strip

A seven-day rail so the whole service week is one glance. Today wears a dot, selection fills ember; the arrows shift a week at a time.

A calendar-page feel: weekday, an oversized date numeral, the month — round chevrons either side and a one-tap Today reset.
